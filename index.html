🔗 Koneksi Repository

🔴 Tidak Terhubung
Token harus memiliki akses 'repo' untuk membaca dan menulis file

📋 Panduan Setup

  1. Buat repository baru di GitHub
  2. Buat Personal Access Token di GitHub Settings > Developer settings > Personal access tokens
  3. Berikan permission 'repo' pada token
  4. Masukkan username, repository name, dan token di atas
  5. Klik 'Test Koneksi' untuk memastikan setup berhasil
